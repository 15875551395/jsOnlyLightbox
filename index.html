Usage

Add the link to the CSS-File to the head of your html-file:

<link rel="stylesheet" href="css/lightbox.css">

Add this before the closing body tag:

<script src="jslightbox.js" type="text/javascript"></script>
<script>
var lightbox = new Lightbox();
lightbox.load();
</script>

A complete init of the lightbox with all options looks like this (click here for full API & options):

var lightbox = new Lightbox();

var lightBoxOptions = {
 responsive: true,
 preload: true,
 loadingImg: true,
 loadingImgSrc: 'img/jslghtbx-loading.gif',
 carousel: true,
 closeOnClick: true,
 hideOverflow: true,
 dimensions: true,
 controls: true,
 animation: 400,
 captions: true,
 onopen: function(){
  console.log('opened')
 },
 onresize: function(){
  console.log('resized')
 },
 onclose: function(){
  console.log('closed')
 },
 onload: function(){
  console.log('loaded')
 }
}

lightbox.load(lightBoxOptions);

Then you just have to add the attribute data-jslghtbx to all the images you want to show in the lightbox:

<img class="jslghtbx-thmb" src="img/lightbox/1.jpg" alt="" data-jslghtbx>

If you want to use captions add the data-jslghtbx-caption attribute

<img class="jslghtbx-thmb" src="img/lightbox/1.jpg" data-jslghtbx-caption="This is my caption text." alt="" data-jslghtbx>

You can also pass an link to another image, i.e. data-jslghtbx='img/lightbox/apple.jpg':

<img class="jslghtbx-thmb" src="img/lightbox/1.jpg" alt="" data-jslghtbx="img/1-big.jpg">

If you want to use multiple images, set the data-jslghtbx-group-attribute:

<img class="jslghtbx-thmb" src="img/lightbox/1.jpg" alt="" data-jslghtbx="img/1-big.jpg" data-jslghtbx-group="group1">
<img class="jslghtbx-thmb" src="img/lightbox/2.jpg" alt="" data-jslghtbx="img/1-big.jpg" data-jslghtbx-group="group1">

Note: The CSS-class jslghtbx-thmb is optional. You can use your own styling if you want.
