jsOnlyLightbox v0.1

Responsive Lightbox written in plain JavaScript

  • No jQuery needed
  • Fully responsive
  • Customizable
  • Lightweight (< 5Kb)
  • IE8+ supported
  • Licensed under MIT, free for commercial & personal use

Use bigsized pictures for both thumbnail and lightbox

Using smallsized pictures for thumbnails and bigsized pictures for lightbox to reduce traffic

Usage

Add the link to the CSS-File to the head of your html-file:

<link rel="stylesheet" href="css/lightbox.css">

Add this before the closing body tag:

<script src="jslightbox.js" type="text/javascript"></script>
<script>
var lightbox = new Lightbox();
lightbox.load();
</script>

Then you just have to add the attribute data-lightbox to all the images you want to show in the lightbox:

<img class="jslghtbx-thmb" src="img/lightbox/1.jpg" alt="" data-lightbox>

You can also pass an link to another image, i.e. data-lightbox='img/lightbox/apple.jpg':

<img class="jslghtbx-thmb" src="img/lightbox/1.jpg" alt="" data-lightbox="img/1-big.jpg">

Note: The CSS-class jslghtbx-thmb is optional. You can use your own styling if you want.
